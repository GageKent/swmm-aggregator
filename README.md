# SWMM Data Aggregator

This project is a one-stop-shop UI for pulling data needed to get SWMM-based stormwater models off the ground. The app will pull elevation data uskng The National Map (TNM) API and merge DEM's before providing to the user.

⬜️ Implement TNM data access

⬜️ Add more user-friendly boundary selection

⬜️ Implement Design Storm retrieval from NOAA
